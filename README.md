# Prometheus DevOps Linter

High-level linter for Prometheus rules and jobs.

## Usage

```
prometheus-devops-linter jobs test-data/jobs/*.*
prometheus-devops-linter rules test-data/rules/*.*
```

## Example

See `.prometheus-linter.yaml` file.
